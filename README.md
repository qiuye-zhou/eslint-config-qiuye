# Introduction

# Usage
```
npm i -D eslint-config-qiuye

yarn add -D eslint-config-qiuye

pnpm add -D eslint eslint-config-qiuye
```



```
// .eslintrc.json

{
    "extends": ["qiuye"]
}
```



## Style

2. Always single quote
3. No trailing commas
